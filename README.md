# DevFolio Portfolio

A personal portfolio website built with **Bootstrap 5, HTML, CSS, and JavaScript**.  
Showcases **skills, projects, services**, and a **responsive design** with an **animated header** and **interactive navbar**.

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- Font Awesome
- SwiperJS

---

## ✨ Features
- Responsive layout for all devices (desktop, tablet, mobile)  
- Animated typing effect in the header (`Designer → Developer → Freelancer`)  
- Interactive navbar with active link indicator  
- Portfolio section showcasing projects with hover effects  
- Progress bars for skills display  
- Contact form with validation  
- Testimonials slider using SwiperJS
